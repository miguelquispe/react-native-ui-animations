## Installation

1. Clone the repository
2. Go to the repo folder with `cd`
3. Run `npm install`
4. Run `npx expo start`
5. Press `a` for open in Android - Press `i` for iOS Simulator

## Stack

[Expo](https://docs.expo.dev/) - SDK ^46
